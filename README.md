## Описание

Программа демонстрирует использование различных контейнеров стандартной библиотеки шаблонов (STL) C++:

- **Последовательный контейнер** – `vector<Money>`
- **Адаптер контейнера** – `queue<Money>`
- **Ассоциативный контейнер** – `multimap<int, Money>`

Тип элементов – пользовательский класс `Money`, представляющий денежную сумму (рубли – `long`, копейки – `int`). Для класса перегружены необходимые операторы: ввод/вывод, сравнение (`==`, `<`), сложение (`+`), присваивание.

### Выполняемые операции (для каждого контейнера)

1. **Задание 3** – найти минимальный элемент и добавить его в конец контейнера.
2. **Задание 4** – удалить элемент по заданному ключу:
   - для `vector` и `queue` ключом является значение типа `Money`,
   - для `multimap` – целочисленный ключ.
3. **Задание 5** – к каждому элементу прибавить сумму минимального и максимального элементов контейнера.

Все операции реализованы в виде глобальных функций, работающих с соответствующими контейнерами.
